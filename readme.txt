Accept donations via Paiement Pro payment gateway using GiveWP.

== Description ==
This plugin will be used to integrate PaiementPro with GiveWP WordPress donation plugin.

= What is PaiementPro? =
[PaiementPro](https://paiementpro.net) is an Intuitive and Intelligent web platform that mainly allows African businesses to accept mobile payments (Orange, MTN, Moov) and electronic payments (Visa Card, MasterCard) *across the continent*, with a Global vision to extend to all other continents.


= What is GiveWP? =
[GiveWP](https://givewp.com) is the highest rated, most downloaded, and best supported donation plugin for WordPress.

= Features =
* Accepts one-off donations via Orange Money
* Accepts one-off donations via MTN Money
* Accepts one-off donations via Moov Money
* Accepts one-off donations via Card Payments

== Installation ==

= Minimum Requirements =

* WordPress 4.8 or greater
* PHP version 5.6 or greater
* MySQL version 5.0.15 or greater
* cURL version 5.40 or higher
* An SSL certificate

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of "PaiementPro for Give", log in to your WordPress dashboard, navigate to the Plugins menu and click "Add New".

In the search field type "PaiementPro for Give" and click Search Plugins. Once you have found the plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our donation plugin and uploading it to your server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
